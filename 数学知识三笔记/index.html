# 高斯消元 O (n^3)

解方程：无解 \ 无穷多解 \ 有唯一解

利用线性代数初等行列变换
1. 把某一行乘一个非零的数
2. 交换某两行
3. 把某行若干倍加到另一行上去

变换之后结果与解的关系:
1. 完美阶梯型 唯一解
2. 不完美阶梯型 0 = 非零 无解
3. 不完美阶梯型 0=0 无穷解

浮点数判断是否为零 需要和 eps 比

算法步骤:

  • 枚举每一列 c
    1. 找到绝对值最大的一行
    2. 将该行换到最上面
    3. 将改行第一个数变为 1
    4. 将下面所有行的第 c 列消成 0

# 卡特兰数
