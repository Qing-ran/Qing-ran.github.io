# 字符串哈希

字符串前缀哈希法

先预处理出来所有前缀的哈希

核心思想：将字符串看成 P 进制数，P 的经验值是 131 或 13331 ，取这两个值的冲突概率低
小技巧：取模的数用 2^64，这样直接用 unsigned long long 存储，溢出的结果就是取模的结果

过程：
1. 看成 p 进制数，每一位看作是 p 进制的数
2. 转换成十进制
3.mod q
4. 映射

注意：
1. 不能映射成 0，从 1 开始
2. 假定无冲突 p=131 || p = 13331 && q=2^64

作用：
可以快速判断，任意两段区间内的字符串是否相同
