高精度

常用分类：

  1. 两个大整数相加（A+B） A和B位数大约为1e6
  2. 两个大整数相减（A-B） A和B位数大约为1e6
  3. 一个大整数乘一个小整数（A*a） len(A)<=1e6 / a<=1e9
  4. 一个大整数除一个小整数(A/a) len(A)<=1e6 / a<=1e9

大整数如何存储：每一位存到数组中（从个位开始存较好）

思路：

前缀和与差分

☆前缀和与差分互为逆运算★

前缀和

一维前缀和

原数组 ：a[1],a[2],a[3]......a[n]
前缀和数组(下标一定要从1开始) ：s[i] = a[1]+a[2]+a[3]+......+a[i]

问题：

  1. 如何求s[i]
    s[0] = 0 (更好的处理边界问题)
    for(int i=1;i<=n;i++) s[i]=s[i-1]+a[i]
  2. 作用
    能快速求出原数组中一段数的和
    求[ l , r ]之间的和 sum = s [ r ] - s [ l - 1]

二维前缀和

快速求子矩阵中的和

S[i, j] = 第i行j列格子左上部分所有元素的和
以(x1, y1)为左上角，(x2, y2)为右下角的子矩阵的和为：
S = S(x2,y2) - S(x2,y1-1) - S(x1-1,y2) + S(x1-1,y1-1)

求 s[ i , j ]

差分

原数组 ：a[1],a[2],a[3]......a[n]
构造：b[1],b[2],b[3]......b[n]
使得：a[i] = b[1]+b[2]+b[3]+......+b[i]

a是b的前缀和，b是a的差分

一维差分

构造方法 ： b[i] = a[i] - a[i-1]

作用
O(1)给区间[l, r]中的每个数(a数组)加上c：B[l] += c, B[r + 1] -= c

二维差分

给以(x1, y1)为左上角，(x2, y2)为右下角的子矩阵中的所有元素加上c：
S[x1, y1] += c, S[x2 + 1, y1] -= c, S[x1, y2 + 1] -= c, S[x2 + 1, y2 + 1] += c
