# 1. 什么是爬虫？

网络爬虫（又被称为网页蜘蛛，网络机器人）就是模拟浏览器发送网络请求，接收请求响应，一种按照一定的规则，自动地抓取互联网信息的程序。
网络爬虫是一种按照一定的规则、自动的抓取万维网信息的脚本或者程序
简单来说：爬虫是用事先写好的程序去抓取网络上的数据，这样的程序叫爬虫
也就是说，爬虫是通过编写程序，模拟浏览器上网，然后让其去互联网上抓取数据的过程

# 2. 爬虫的分类

1. 通用爬虫
抓取系统重要组成部分。抓取的是一整张页面数据。
2. 聚焦爬虫
建立在通用爬虫基础上，抓取的是页面中特定的局部内容。
3. 增量式爬虫
监测网站中数据更新的情况，只会抓取网站中最新更新出来的数据。

# 3. 反爬机制与反反爬策略

# 3.1 反爬机制

门户网站可以通过制定相应的策略或者技术手段，防止爬虫程序进行网站数据的爬取。

# 3.2 反反爬策略

爬虫程序可以通过制定相关的策略或者技术手段，破解门户网站中具备的反爬机制，从而可以获取门户网站中相关的数据。

比如： robots.txt 协议（君子协议）
