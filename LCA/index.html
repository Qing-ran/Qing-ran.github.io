# 1. 向上标记法 O (n)

# 2. 倍增法 O (mlogn)

  • 预处理 fa[i,j] 表示从 i 开始，向上走 2^j 步所能走到的节点。 0<=j<=logn
  • depth [i] 表示深度
  • 查询 O (logn) 预处理 O (nlogn)
  • 哨兵： 如果从 i 开始跳 2^j 步会跳过根节点那么 fa[i,j]=0,depth[0]=0

步骤：

fa[6,0] = 4 fa[6,1] = 2 fa[6,2]=NULL

在线算法：每读入一个询问就输出
离线算法：把所有询问读入，最后统一输出
